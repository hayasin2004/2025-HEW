# Plasmo Translation Tool 使い方ガイド

## 1. セットアップ（初回のみ）

### 拡張機能の読み込み
開発モードで拡張機能をChromeにインストールする手順です。

1. Chromeブラウザを開き、URLバーに `chrome://extensions` と入力して移動します。
2. 画面右上の **「デベロッパーモード」** をオンにします。
3. 左上の **「パッケージ化されていない拡張機能を読み込む」** (Load unpacked) をクリックします。
4. 本プロジェクト内の以下のフォルダを選択します：
   `build/chrome-mv3-dev`
   ※ `npm run dev` を実行してビルドが完了している必要があります。

### APIキーの設定
本ツールは外部サービスを使用するため、以下のファイルにご自身のAPIキーを設定する必要があります。

1. **Firebase (認証・データベース)**
   - ファイル: `lib/firebase.ts`
   - 設定項目: `firebaseConfig` オブジェクト内の各値を、Firebaseコンソールから取得した値に書き換えてください。

2. **Google Translate API (翻訳)**
    - ファイル: `lib/api/translate.ts`
    - 設定項目: 現在はモック（ダミー）実装になっています。実際のAPIを使用する場合は、`fetch` 関数などでGoogle Cloud Translation APIを呼び出すコードに書き換えてください。

## 2. 基本的な使い方

アイコンをクリックしてポップアップを開きます。

### 🅰️ 翻訳 (TRANS)
1. **Source Text** に翻訳したいテキストを入力します。
2. 言語選択プルダウンで翻訳先の言語を選びます。
3. **「TRANSLATE ->」** ボタンをクリックします。
4. 結果が画面下部に表示されます。
   - ログインしている場合、翻訳結果は自動的に「履歴」に保存されます。

### 🅱️ 履歴 (HISTORY)
1. 上部ナビゲーションの **「HISTORY」** をクリックします。
2. 過去の翻訳履歴がリスト表示されます。
   - ※履歴機能を利用するには、Firebase Authenticationでのログイン実装が必要です（現在は仮実装）。

### 🆎 テンプレート (TEMPLATES)
1. 上部ナビゲーションの **「TEMPLATES」** をクリックします。
2. 保存済みの定型文が表示されます。
   - よく使うメールの挨拶などを登録しておくと便利です。

## 3. 開発・トラブルシューティング

- **コードの変更**: VSCode等でコードを編集して保存すると、自動的にリビルドが走ります。
- **反映されない場合**: 拡張機能管理画面 (`chrome://extensions`) で、該当する拡張機能の更新ボタン（回転矢印アイコン）を押してください。
